# alfred-digit-conversion-workflow


## Usage

Decimal: `hex 1234` =>
* 4d2 (Hexadecimal)
* 1234 (Decimal)
* 10011010010 (Binary)

Hexadecimal: `hex 0x1234` =>
* 1234 (Hexadecimal)
* 4660 (Decimal)
* 1001000110100 (Binary)

Octal: `hex 012` =>
* a (Hexadecimal)
* 10 (Decimal)
* 1010 (Binary)

Binary: `hex b1011` =>
* b (Hexadecimal)
* 11 (Decimal)
* 1011 (Binary)

Press Enter on item to copy to clipboard.
